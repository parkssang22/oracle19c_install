## CDB와 PDB 간의 세션 전환 및 객체 수 비교 확인
```bash
SQL> show con_name

CON_NAME
------------------------------
CDB$ROOT

SQL> select count(*) from dba_objects;

  COUNT(*)
----------
     72374

SQL> alter session set container=ORCLPDB1;

Session altered.
SQL> show con_name

CON_NAME
------------------------------
ORCLPDB1

SQL> select count(*) from dba_objects;

  COUNT(*)
----------
     72356
```
- CDB, PDB의 오브젝트 수가 차이난다
