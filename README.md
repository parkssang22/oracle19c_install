# oracle19c_install

## - RPM이란?
  - 오라클이 19c부터 지원하는 기능으로 하나의 명령어로 11g를 설치할 때 했던 파라미터 설정과 유저 및 그룹 생성, 권한 설정과 오라클 소프트웨어 설치까지 자동화한다
