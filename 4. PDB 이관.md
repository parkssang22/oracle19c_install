## PDB 이관하는 방법
```bash
SQL> show con_name

CON_NAME
------------------------------
CDB$ROOT
```

## PDB를 내린다
```bash
SQL> alter pluggable database ORCLPDB1 close immediate;
```

## PDB를 xml형식으로 /home/oracle/ 경로로 내보낸다
```bash
SQL> alter pluggable database ORCLPDB1 unplug into '/home/oracle/orclpdb1.xml';
```

## PDB를 삭제한다
```bash
SQL> drop pluggable database ORCLPDB1 keep datafiles;

플러그인할 수 있는 데이터베이스가 삭제되었습니다.

SQL> show pdbs;

    CON_ID CON_NAME                       OPEN MODE  RESTRICTED
---------- ------------------------------ ---------- ----------
         2 PDB$SEED                       READ ONLY  NO
```

## xml형식으로 PDB를 생성한다
- 서로 다른 서버라면 copy 사용
```
SQL> create pluggable database ORCLPDB1
  2  using '/home/oracle/orclpdb1.xml'
  3  nocopy;

플러그인할 수 있는 데이터베이스가 생성되었습니다.
```

## PDB로 접속한다
```
SQL> alter pluggable database ORCLPDB1 open;

플러그인할 수 있는 데이터베이스가 변경되었습니다.

SQL> show pdbs;

    CON_ID CON_NAME                       OPEN MODE  RESTRICTED
---------- ------------------------------ ---------- ----------
         2 PDB$SEED                       READ ONLY  NO
         3 ORCLPDB1                       READ WRITE NO
```
